# Robo-AI-recorded

---

## Course Overview

This repository contains a structured guide for topics in Python, Artificial Intelligence, Machine Learning, and related fields. The content is divided into **Basic** and **Advanced** sections for each topic.

---

## Topics Covered

| **Topic Name**              | **Basic**                                                                                                                                 | **Advanced**                                                                                                           |
|-----------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| **Python 101**              | - Why Python? Intro to Google Colab<br>- Python basics (syntax, datatypes)<br>- Conditional statements, Collections, Loops<br>- String manipulation (slicing, indexing, formatting, splitting, joining)<br>- Classes, objects, Importing Libraries<br>- Python examples | - Object-Oriented Programming (OOPs)<br>- Classes & Objects, Inheritance, Polymorphism & Scope<br>- Keywords (`self`, `with`, `open`, etc.)<br>- Functions and Methods<br>- Creating custom Python libraries |
| **Artificial Intelligence** | - Theoretical understanding of AI, ML, and DL<br>- Concepts of Search, Knowledge, and Uncertainty                                      | - Hands-on coding for:<br>&nbsp;&nbsp;- Depth-first Search<br>&nbsp;&nbsp;- Breadth-first Search<br>&nbsp;&nbsp;- Greedy Best-first Search<br>&nbsp;&nbsp;- A* Search |
| **Machine Learning**        | - Overview of Regression and Classification Problems                                                                                  | - Hands-on applications, mathematical intuition, and project development for:<br>&nbsp;&nbsp;- Linear Regression<br>&nbsp;&nbsp;- Logistic Regression<br>&nbsp;&nbsp;- Decision Trees |
| **Reinforcement Learning**  | - Introduction to RL<br>- Algorithms in Discrete and Continuous Spaces<br>- Q-learning, PPO                                           | - Hands-on RL:<br>&nbsp;&nbsp;- Environment definition<br>&nbsp;&nbsp;- Simulation of a toy-world                      |
| **Artificial Neural Networks** | - History of AI<br>- Biological Neurons vs Artificial Neurons<br>- Perceptrons, Single-layer architecture<br>- Multi-layer networks | - Forward and backward propagation<br>- Hands-on programming a Perceptron from scratch                                 |
| **Deep Learning**           | - Activation Functions<br>- Types of Deep Neural Network Architectures<br>- Encoders and Decoders                                    | - Hands-on Deep Learning Architecture Development using PyTorch/TensorFlow                                            |

| **Image Processing**        | - What is an image (human vs computer)?<br>- Image enhancement techniques, color correction                                           | - Hands-on:<br>&nbsp;&nbsp;- Image sharpening<br>&nbsp;&nbsp;- Edge detection<br>&nbsp;&nbsp;- Noise reduction<br>&nbsp;&nbsp;- Feature extraction |
